Overview
A fully functional NFT Marketplace built with React, featuring dynamic data fetching, animated UI components, and a clean, modern layout inspired by real NFT platforms. Users can browse collections, explore new items, view top sellers, and dive into detailed author and item pages — all powered by a responsive API layer.
